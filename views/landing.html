EOS icons offers an easily customizable animated icons. These
Icons use only CSS animations and nothing more which make
things a lot easier when using these icons in your own projects.
Also they run on the GPU so the frame rate is much higher
than that of Javascript animations. When it comes to CSS
animations they also have the advantage of not causing
reflows and redraws unlike modifying elements via
Javascript would.
